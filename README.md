## About Larapus

Larapus is a library management system built with Laravel 7

## Features

- User management (Admin, Librarian, and Member).
- Book management stock.
- Bookshelf.
- Search Book.
- Book List (On progress).
- Penalty (On progress).
- Report (On progress).
